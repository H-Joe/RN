# 学习React Native所练习的Demo

###Usage
```
yarn install
react-native link
```
>最后编写一个综合案例，基于豆瓣开放平台接口

>受设备所限，目前只适配iOS

### 技术栈
`react-native`+`redux`+`redux-thunk`+`react-navigation`+`react-redux`+`react-native-vector-icons`
